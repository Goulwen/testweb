Les commandes

/hello : permet d'afficher helloworld
/download : permet de télécharger un fichier
/pull : permet de voir un compteur qui diminue jusqu'à zéro
/openSession.html : ouvre une session
/bug.html : réorientation si erreur avec /bug.html?echec=true
/admin/secret.html : accès sécurisé pour la page secret.html

/hello.jsp : permet d'afficher message basique avec un fichier jsp
/entreeMVC.html permet de générer un nombre de personne avec MVC
